We use [Google Material Icons](https://fonts.google.com/icons) (Also called Google Material Symbol the new Google icons library) through the `<ts-icon>` component. The examples below show commonly used icons in our design system.

## Usage

#### Web Components

```js
<ts-icon name="home"></ts-icon>
```

#### React

```jsx
import { TsIcon } from '@truvantis/design-system/react';

<TsIcon name="home"></TsIcon>;
```
