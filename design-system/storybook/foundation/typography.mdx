The brand font used in our design system is **TUV SUD Pro**. It is used for all text elements including headings,
paragraphs, buttons, and labels.

Should import the font in your project to ensure proper rendering. At the entry point of your application, include the following line:

```jsx
import '@tuvsud/design-system/theme/fonts.css';
```

### Font Family

The primary font family used is:

```css
font-family: 'TUV SUD Pro', system-ui, sans-serif;
```
